![Version](https://img.shields.io/badge/version-1.6.0-blue.svg)

![Platform](https://img.shields.io/badge/platform-Fire%20TV%20%7C%20Android%20TV%20%7C%20Web-orange.svg)
![Tech Stack](https://img.shields.io/badge/tech-HTML%20%7C%20CSS%20%7C%20JS%20%7C%20Firebase%20%7C%20HuggingFace-purple.svg)

**Il Covo di Nello** - Portale web per download e installazione di APK su dispositivi **Amazon Fire TV Stick** e **Android TV**.

Pannello admin protetto, cron job automatici per aggiornamenti, notifiche email/Telegram.

## Stack Tecnologico

- **Frontend**: HTML5, CSS3, Vanilla JS
- **Backend**: Node.js + Express
- **Database**: Firebase Realtime Database
- **Auth**: Firebase Authentication
- **Hosting**: Hugging Face Spaces (Docker)
- **Cron**: node-cron (in-process)
