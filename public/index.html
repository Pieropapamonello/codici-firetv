🔧 Guide avanzate Fire TV & EasyProxy
🔋 Fire TV — Disabilitare risparmio energetico per EasyProxy

1. Sblocca Opzioni Sviluppatore

Impostazioni → La mia Fire TV → Informazioni → clicca 7 volte su "Numero di serie" → Opzioni Sviluppatore → attiva Debug ADB e Debug ADB di rete

2. Trova l'IP della Fire TV

Impostazioni → La mia Fire TV → Informazioni → Rete → annota l'IP (es. 192.168.1.XX)

3. Installa Termux sul telefono

Scarica da F-Droid (NON Play Store): f-droid.org/com.termux
⚠️ Telefono e Fire TV devono essere sulla stessa rete Wi-Fi

4. Installa ADB su Termux

pkg update && pkg install android-tools -y

5. Connettiti alla Fire TV

adb connect 192.168.1.XX
👉 Sostituisci l'IP. Accetta il pop-up sulla TV → "Consenti sempre"

6. Esegui i comandi

adb shell dumpsys deviceidle whitelist +com.mediaflow.proxy.tv
adb shell pm grant com.mediaflow.proxy.tv android.permission.REQUEST_IGNORE_BATTERY_OPTIMIZATIONS

7. Verifica

adb shell dumpsys deviceidle whitelist
✅ Dovresti vedere com.mediaflow.proxy.tv nell'elenco

↩️ Per annullare

adb shell dumpsys deviceidle whitelist -com.mediaflow.proxy.tv
adb shell pm revoke com.mediaflow.proxy.tv android.permission.REQUEST_IGNORE_BATTERY_OPTIMIZATIONS

📌 Permanente ai riavvii. Si resetta solo con aggiornamenti Fire OS — in quel caso ripeti solo i punti 5 e 6.

🌐 Esporre EasyProxy su Internet via Cloudflare Tunnel — Parte 1 (link temporaneo)

1. Installa Termux sul telefono

f-droid.org/com.termux (NON Play Store)

2. Installa cloudflared

pkg update && pkg install cloudflared -y

3. Avvia il tunnel temporaneo

cloudflared tunnel --url http://localhost:7860
✅ Apparirà un link tipo https://qualcosa-random.trycloudflare.com
⚠️ Il link cambia ad ogni riavvio
🌐 Cloudflare Tunnel — Parte 2 (link fisso permanente, gratis)

4. Registra un dominio gratuito

domain.digitalplat.org → scegli un nome tipo mioproxy.dpdns.org

5. Crea un account Cloudflare gratuito

dash.cloudflare.com → + Add → Connect a domain → inserisci il tuo dominio DigitalPlat → Continue

6. Copia i nameserver Cloudflare

Cloudflare ti mostrera' 2 nameserver tipo:
daisy.ns.cloudflare.com fred.ns.cloudflare.com

7. Aggiorna i nameserver su DigitalPlat

DigitalPlat → Domain List → sostituisci i nameserver con quelli di Cloudflare → salva
⏳ Aspetta l'email di conferma da Cloudflare (10 min - 2 ore)

8. Crea il tunnel fisso

Cloudflare → menu sinistra → Networking → Tunnels → Create a Tunnel → dai un nome (es. pippo) → Save

9. Collega Termux al tunnel

Nella pagina del tunnel seleziona: OS Debian, Architettura arm64
Copia il comando sotto "Or, run tunnel (manual)" e incollalo su Termux
✅ Il tunnel diventera' Healthy

10. Aggiungi il dominio al tunnel

Cloudflare → Networking → Tunnels → tab Routes → + Add route → Public Hostname:
  • Subdomain: lascia vuoto (o scrivi proxy)
  • Domain: tuosito.dpdns.org
  • URL: http://localhost:7860
✅ Il tuo link fisso sara' https://tuosito.dpdns.org per sempre
⚙️ Cloudflare Tunnel — Parte 3 (gestione manuale)

Salva il comando per avviarlo facilmente

echo "INSERIRE LINK DEL TUNNEL OTTENUTO nel punto 9" > ~/tunnel.sh chmod +x ~/tunnel.sh

Aggiungi un alias per avviarlo velocemente

echo "alias tunnel='bash ~/tunnel.sh'" >> ~/.bashrc source ~/.bashrc

Avviare il tunnel

bash ~/tunnel.sh
oppure semplicemente tunnel

Stoppare il tunnel

Premi Ctrl+C su Termux oppure chiudi la sessione Termux
🚀 Cloudflare Tunnel — Parte 4 (avvio automatico al boot del telefono)

⚠️ Avviare prima l'app del proxy, scorrere e mettere la spunta su "Autostart on boot"

1. Installa Termux:Boot da F-Droid

f-droid.org/com.termux.boot
⚠️ Aprilo almeno una volta dopo l'installazione

2. Crea lo script di avvio automatico

pkg install nano -y mkdir -p ~/.termux/boot nano ~/.termux/boot/start-tunnel.sh
Scrivi queste 3 righe nel file:
#!/data/data/com.termux/files/usr/bin/sh termux-wake-lock bash ~/tunnel.sh
Salva: Ctrl+X → Y → Invio

3. Rendi lo script eseguibile

chmod +x ~/.termux/boot/start-tunnel.sh

4. Verifica che il file sia corretto

cat ~/.termux/boot/start-tunnel.sh
✅ Devono apparire esattamente le 3 righe sopra

5. Disabilita ottimizzazione batteria

Impostazioni Android → App → fai "Nessuna restrizione" per:
  • ✅ Termux
  • ✅ Termux:Boot
  • ✅ EasyProxy

Risultato finale: Ad ogni riaccensione del telefono → 🔒 Wake lock attivo (Termux rimane sveglio in background) → 🚀 Tunnel Cloudflare parte automaticamente.
📲 EasyProxy va avviata manualmente oppure impostata su "avvia all'avvio" nelle sue impostazioni.

📶 Fire TV — Configurare IP/DNS manualmente

Metodo direttamente dalla Fire TV

  1. Vai su Impostazioni
  2. Entra in Rete
  3. Seleziona la tua rete Wi-Fi
  4. Premi il tasto con le tre linee (☰) sul telecomando → Avanzate
  5. Se disponibile, scegli Configura IP → Manuale
A questo punto devi inserire:
  • Indirizzo IP (es. 192.168.1.50)
  • Gateway (di solito 192.168.1.1)
  • Prefisso di rete (di solito 24)
  • DNS 1 (es. 8.8.8.8)
  • DNS 2 (es. 8.8.4.4)
🦴 Aggiungi ai preferiti: se hai una Fire TV, tieni premuto il tasto ☰ (3 lineette) sul telecomando.
